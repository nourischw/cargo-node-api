# cargo-node-api v1

## Install ##
- git [https://git-scm.com/] (with git bash)
- node.js (v8.1.0) - [https://nodejs.org/en/blog/release/v8.1.0/]

## Project Control ##
```sh
open git bash
cd cargo-node-api
npm install
npm install -g
```

## Run ##
```sh
open git bash
cd cargo-node-api
NODE_PATH=. LOCAL=1 node app.js
```

## Example ##
```sh
http://localhost:3000/v1/test
```
